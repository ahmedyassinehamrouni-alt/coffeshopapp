# BrewDesk ☕ — Coffee Shop Order Management

Flutter + Firebase order management system for coffee shops.

---

## Tech Stack

| Layer | Technology |
|---|---|
| Framework | Flutter 3.x |
| State management | Riverpod 2 (code-gen) |
| Navigation | GoRouter 14 |
| Backend | Firebase (Firestore, Auth) |
| Models | Freezed + json_serializable |

---

## Quick Start

### 1. Prerequisites
```bash
flutter --version   # >= 3.22
dart --version      # >= 3.3
```

### 2. Firebase Setup
```bash
# Install FlutterFire CLI
dart pub global activate flutterfire_cli

# Create a Firebase project at console.firebase.google.com, then:
flutterfire configure --project=YOUR_PROJECT_ID

# Enable in Firebase Console:
# - Authentication → Email/Password
# - Firestore Database (start in production mode)
# - Storage (optional, for menu images)
```

### 3. Install dependencies
```bash
flutter pub get
```

### 4. Run code generation
```bash
dart run build_runner build --delete-conflicting-outputs
```

### 5. Deploy Firestore rules & indexes
```bash
firebase deploy --only firestore
```

### 6. Seed initial admin user
```
In Firebase Auth: create a user manually.
In Firestore, create: staff/{uid} with:
  { name: "Admin", email: "...", role: "admin", isActive: true }
```

### 7. Run the app
```bash
flutter run
```

## Code Generation

Regenerate after editing any `@freezed`, `@riverpod`, or `@JsonSerializable` class:
```bash
dart run build_runner build --delete-conflicting-outputs
# or watch mode during development:
dart run build_runner watch --delete-conflicting-outputs
```
